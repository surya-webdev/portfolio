My Role



Firstly , Implemented robust user authentication using Supabase, a backend-as-a-service (BaaS) solution. Supabase provides secure login functionality and fine-grained access control, ensuring only authorized users can access the application and manage booking data.

Secondly, I added a remote state management library like tanQuery can be utilized to efficiently manage application state. This ensures data consistency across different components of the application, such as user interface elements that reflect real-time booking information.

Thirdly, I honed my React skills, focusing on database communication and implementing CRUD operations for bookings and cabins, as well as updating details.

Fourthly, I added a key feature that allows for the addition of booking customer details and check-in/check-out functionality.

Fifthly, to boost sales, I implemented a visual dashboard. This dashboard provides data visualization for sales and customer influx using the powerful Rechart library.

Finally, By implementing these features, Istay offers a comprehensive solution for hotel booking management, enhancing efficiency and user experience for administrators.
